Professional Work

Harley School Blog

The Harley School has been a long time client of EagleDream and has come to expect the best not only for their students, but for their web experience as well. In addition to adding a Calendar feature to their main site that dynamically creates pages for iCal and Google Calendar events, I was able to work on a second site for them. The Harley Blog is a place for teachers to have better communication and transparency with parents as to what’s being achieved in the classroom. The project exposed me to the infrastructure and security challenges that come along with creating password protected blogs for several separate blogs.

Just Water

While working with JUST Water at EagleDream Technologies I was tasked with creating a scalable landing page for users across the world, directing them to the correct version of the site depending on their region. The design was simple, but there were no mobile views or indications of how the page should display on different screen sizes. I created the file structure, gathered the necessary media assets, and implemented queries that scale the fonts and content to fit phones, tablets, and desktops.

School-Related Projects

Octuv Music

Using stark contrast of popping colors between sections that run the full length of the screen, I’ve created a clean-looking site that scales beautifully and looks great in browsers and mobile devices. Octuv Music is an app idea that a group of students and I developed, making wireframes for mobile pages and eventual mock-up screenshots of what the finish app would look like. I also shot and edited a commercial that explains the use of our app idea which was very fun to make. The outcome was sleek, uncluttered, and provided the appropriate “download” links and social media plugs.

Hometown Site

With a focus on easy navigation using bright images to navigate users through the pages, my Hometown website shows off fancy JavaScript dropdowns and well styled buttons. The purpose of the site is to provide easy-to-find information about the town I grew up in, and is intended for users of any age. All content, banners, and graphics were designed and made by myself.

Red Barn Climbing

Intended as a proper website for the Red Barn climbing facility at R.I.T., I designed and implemented a colorful page with large navigational icons to allow easy access to all of the information on the site. This project exercised making content for a real business, and is in the process of being used for the official Red Barn Climbing web site.

Choose-Your-Own Adventure

Though the styling on this site is lack luster, this choose-your-own adventure site exhibits advance use of JavaScript to read a JSON file and dynamically create elements on the page. It demonstrates understanding of the DOM and has reusable functions that make the various html elements necessary to create the site. Changing choices in the story will delete child elements and replace them with new information based on these story decisions. It showcases my ability to use JavaScript functions to read files and create web elements.
